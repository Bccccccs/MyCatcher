Problem Description

You are given three integers N, A, and B.

Consider a repeating cycle of length A + B. In each cycle, the first A positions are counted, and the next B positions are not counted.

For the integers from 1 to N inclusive, determine how many integers fall within the counted positions of this repeating pattern.

⸻

Input Format

The input contains three integers:

N A B

⸻

Constraints

1 ≤ N ≤ 10^9
1 ≤ A, B ≤ 10^9

⸻

Output Format

Output a single integer representing the number of integers from 1 to N that fall within the counted positions of the pattern.
