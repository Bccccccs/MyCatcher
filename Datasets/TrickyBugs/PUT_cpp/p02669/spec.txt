Problem Description

For each test case, you are given an integer n and four operation costs a, b, c, and d.

You want to reduce n to 0 with minimum total cost using the following operations:
	1.	Decrease or increase the current value by 1, with cost d for each unit change.
	2.	If the current value is divisible by 2, you may replace it with x / 2 at cost a.
	3.	If the current value is divisible by 3, you may replace it with x / 3 at cost b.
	4.	If the current value is divisible by 5, you may replace it with x / 5 at cost c.

Before applying a division operation, you may increase or decrease the current value by several units so that it becomes divisible by 2, 3, or 5, paying d for each unit changed.

Your task is to find the minimum total cost to reduce n to 0.

Input Format

The first line contains a single integer:

t

Then t test cases follow.

Each test case contains five integers:

n a b c d

Constraints

1 ≤ t ≤ 10^4
0 ≤ n ≤ 10^18
1 ≤ a, b, c, d ≤ 10^18

Output Format

For each test case, output a single integer representing the minimum total cost to reduce n to 0.
