Problem Description

You are given the results of N submissions in a programming contest.

Each submission result is one of the following four strings:

AC
WA
TLE
RE

Count how many times each result appears.

Input Format

The first line contains a single integer:

N

Each of the next N lines contains a string:

S_i

Constraints

1 <= N <= 10^5
Each S_i is one of: AC, WA, TLE, RE

Output Format

Output exactly four lines in the following format:

AC x count_AC
WA x count_WA
TLE x count_TLE
RE x count_RE
